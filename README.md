# ReinigungBam - Professionelle Gebäudereinigung in Bamberg

Dies ist eine moderne, statische Website für ein fiktives Reinigungsunternehmen in Bamberg.
Das Projekt dient als Portfolio-Referenz, um die Umwandlung einer dynamischen WordPress-Instanz in eine hochperformante statische Website zu demonstrieren.

## Features
- **Responsives Design:** Optimierte Darstellung auf Smartphones, Tablets und PCs.
- **Leistungsübersicht:** Detaillierte Darstellung der Reinigungsdienste (Büro-, Glas-, Wohnungs- und Baureinigung).
- **Über Uns Seite:** Informationen über das Unternehmen und die Firmenwerte.
- **Blog-Sektion:** Bereich für aktuelle Nachrichten und Reinigungstipps.
- **Kontakt:** Integriertes Kontakt-Layout für Kundenanfragen.
- **Performance:** Extrem schnelle Ladezeiten durch statische Bereitstellung (keine Datenbankabfragen beim Seitenaufruf).
